# HEARTBEAT.md — Periodic Checks

## Check Rotation (pick 1-2 per heartbeat)

### 📧 Email Scan
- Check for unread messages in the last 2 hours
- Flag anything from leadership, direct reports, or key clients
- Summarize only if something needs attention

### 📅 Calendar
- Check next 24 hours for upcoming events
- Alert if meeting < 2 hours away
- Note any conflicts

## Rules
- **Quiet hours (11pm-7am):** HEARTBEAT_OK unless urgent
- **If nothing notable:** HEARTBEAT_OK
- **Don't be noisy:** Max 2-3 proactive messages per day
